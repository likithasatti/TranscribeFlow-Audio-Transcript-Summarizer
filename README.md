🎙️ TranscribeFlow – Audio Transcript Summarizer
📌 Overview
TranscribeFlow is a web-based application that converts audio into text and generates a concise summary using AI. Users can upload audio files or record audio directly, making it easy to extract meaningful information from spoken content.

🚀 Features
🎤 Audio to Text Conversion
📝 Automatic Text Summarization
🔐 User Login & Registration
📁 Upload Audio Files (MP3, WAV)
🎙 Record Audio from Browser
📄 Display Transcript & Summary
⬇ Download Results (TXT / JSON)
💡 Simple & User-Friendly UI

⚙️ How It Works
User logs into the application
Uploads or records audio
Audio is processed by the backend
Speech-to-text conversion is performed
Text summarization is applied
Transcript and summary are displayed
User can download the results
